\# Projet AlgoBox – Travail collaboratif avec GitHub



\## Description

Ce projet a été réalisé dans le cadre d’un TP visant à apprendre l’utilisation de GitHub

pour travailler en binôme sur des algorithmes AlgoBox.



\## Contenu

\- Dossier `src/` : contient les fichiers AlgoBox (.algo)

\- addition.algo : addition de deux nombres

\- moyenne.algo : calcul de la moyenne de deux notes



\## Exécution

1\. Ouvrir AlgoBox

2\. Fichier → Ouvrir

3\. Sélectionner un fichier `.algo` dans le dossier `src`

4\. Lancer l’exécution
